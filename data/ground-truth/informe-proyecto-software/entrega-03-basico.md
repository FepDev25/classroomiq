# Proyecto de salas de estudio

**Autor (ficticio):** Estudiante C — Grupo 1

## Problema

El proyecto es sobre las salas de estudio de la biblioteca. A veces hay problemas con
las reservas. Queremos hacer una página para reservar las salas y que sea más fácil.

Cosas que tiene que hacer:
- Reservar salas.
- Ver las salas.
- Que funcione bien.

## Arquitectura

El sistema usa React para el frontend, Spring Boot para el backend, PostgreSQL para la
base de datos, y también usamos Docker. Estas tecnologías son las que vimos en clase y
son muy usadas en la industria. Con esto se construye la aplicación de reservas.

## Decisiones

Usamos React porque es muy popular y tiene muchos componentes. Usamos Spring Boot porque
es un framework muy completo y conocido. PostgreSQL porque es gratis y potente. Docker
porque sirve para desplegar.

## Conclusión

Con este proyecto aprendimos a hacer una aplicación web para reservar salas usando varias
tecnologías. El sistema queda funcionando para la biblioteca.
